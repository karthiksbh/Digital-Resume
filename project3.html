Smart Shopper - Chrome Extension

  • Source code
  • Download Extension

Tech Stack Used: Flask, BeautifulSoup, HTML, CSS and Javascript

Developed a Chrome extension enabling users to compare prices across Amazon, Flipkart, and other e-commerce websites.

  • Uses Flask, BeautifulSoup, and Requests for web scraping, JavaScript for browser interaction, and HTML/CSS for user interface
  • Helps users to find the best deals and save money while shopping online
