Built an Ecommerce Website using Django Framework

  • Source code

Tech Stack Used: Django, HTML, CSS, Javascript, SQLite

The project includes an online shopping web application which provides the users with a list of products available for purchase & helps in purchasing them. The application allows the users to login to their account, place order, view their orders, add reviews on the products etc.

  • For the front end, I have used HTML, CSS, Bootstrap, and some basic JavaScript.
  • For the backend, I have used Django Framework and SQLite Database to store the respective data.
  • The main objective of this project was to learn about the backend functionality, databases and to understand how this server-side application is integrated with the client-side website.
