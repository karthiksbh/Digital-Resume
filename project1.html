Built backend for the IETE Recruitment Portal (Quiz App)

  • Live Demo
  • Source code

Tech Stack Used: Django Rest Framework and SQLite Database

Includes a quiz application built on the Django Rest Framework that enables users to test their knowledge in the area of their interest.

  • The portal is highly responsive and includes various security features like detecting Tab switch, JWT authentication for every API request , Auto submission on test timeout etc.
  • The system automatically verifies the responses and assigns the final quiz score
  • The portal was used by more than 300 students to give test.
